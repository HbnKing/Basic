Java开发环境搭建

1.    JDK和JRE

JRE是Java运行环境，JRE主要是由JVM和Java核心类库类库组成的。如果想要运行Java程序，必须要Java的运行环境。
JDK是Java开发工具包，是提供给Java开发人员使用的，它里面包含了Java开发工具，也包括JRE，也就是说，JDK中也包含了JVM和Java核心类库。如果我们安装了JDK，就可以不用再单独按照JRE了。
简单地说，我们使用JDK来开发Java程序，然后把程序交给JRE运行。

2.    安装JDK

2.1.  下载JDK

1)          打开连接地址https://www.oracle.com
2)          鼠标移到Menu-->Downloads-->Java-->All Java Downloads
3)          鼠标点击All Java Downloads打开一个新页面
4)          点击“java Downloads”，打开JDK下载页面
5)          这就是打开的JDK的下载页面，通过当前页面下载的JDK是最新版本的JDK
6)          点击上图中的下载图标，打开JDK 8  的下载页面
7)          往下拉，看到
8)          在JDK的下载位置看到“You must accept the Oracle Binary Code License Agreement for Java SE to download this software.”你必须接受Java SE的Oracle二进制代码许可协议下载这个软件。选择“Accept License Agreement”，接受许可协议，就可以根据需要开始下载JDK。
9)          当然，在实际开发的过程中，并不建议使用最新的JDK版本，因为最新版本可能不稳定，所以一般在开发中使用的都是早期的稳定版本。所以我们查找历史版本。点击这个页面的“Downloads”，回到JDK 8的初始打开页面。
回到这个页面
10)      往下拉，找到“Java Archive”，点击右边的“Downloads”
打开新页面，往下拉，就可以看到很多版本的JDK，选择需要的进行下载，不过在这个页面选择版本下载时，需要登录。

2.2.  JDK的安装

1)          这里使用的JDK是：jdk-7u80-windows-x64.exe，双击这个文件
2)          打开了JDK的安装向导，点击“下一步”
3)          在这个窗口里面提示了要安装的有开发工具、公共源码、公共jre。可以点击“更改”修改JDK的安装位置，然后“下一步”
4)          正在进行安装，安装的时间会稍微长一些
5)          询问是否要安装JRE，因为JDK中包含了JRE，所以不需要额外的安装JRE了，点“取消”
6)          弹出对话框，询问是否确实要取消安装，选择“是”
7)          这时候开发工具包就安装完成了，点“关闭”。

2.3.  配置环境变量

1)          JDK安装完成之后，开始进行环境变量的配置，需要配置的环境变量有3个：JAVA_HOME,PATH,CLASSPATH。这三个变量大小写无所谓。
变量设置参数如下：
变量名：JAVA_HOME
变量值：D:\Program Files\DevelopTool\Java\jdk1.7.0_80   //  要根据自己的实际路径配置
变量名：Path
变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
变量名：CLASSPATH
变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;         //记得前面有个"."
2)          现在开始配置环境变量，右键单击“计算机”，选择“属性”，在打开的系统窗口上单击“高级系统设置”
3)          在打开的窗口中选择“高级”-->“环境变量”
4)          就打开了环境变量窗口
5)          在“系统变量”的这一块进行编辑，先新建JAVA_HOME变量，JAVA_HOME的值就是JDK的安装路径（D:\Program Files\DevelopTool\Java\jdk1.7.0_8）
6)          接着配置Path变量，Path变量一般在系统中都有配置，所以
a)            找到Path变量
b)          双击它或者单击它之后点编辑
c)            在Path后面添加要配置的信息：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;，如果原先的Path变量后没有分号，要记得添加分号进行分隔
d)          在Path的配置中点“确定”
7)          最后，配置CLASSPATH变量，它的值是：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;    记得前面有个"."
8)          3个环境变量都配置好了，点击“确定”，回到系统属性窗口，再点“确定”即可。
    
接下来，我们来测试JDK是否配置成功。
1)          在单击开始菜单，在搜索栏中输入cmd，找到命令提示符，选中它
2)          打开了命令提示符窗口，在窗口中分别输入java -version、javac -version，输出正确的版本信息，即JDK的安装及配置成功了。
  

3.    编写第一个java程序

编写java程序的步骤：
（1）将Java代码编写到扩展名为.java的文件中。
（2）通过javac命令对该java文件进行编译。
（3）通过java命令对生成的class文件进行运行。
动手操作编写代码：
1)          先在桌面建一个文件夹day01，在day01中创建一个文本文件，文件名叫做HelloWorld.java。后缀名是.java，并在这个文件中编写java代码，代码如下：
public  class  HelloWorld{
              public  static  void  main(String[] args) {
                           System.out.println("Hello World !");
             }
}
2)          开始菜单，输出cmd，打开命令提示符
3)          打开day01目录
4)          输入javac HelloWorld.java，进行编译
5)          输入java HelloWorld，运行
6)          运行结果，输出了：Hello World !
